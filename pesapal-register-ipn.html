Pesapal IPN Registration

Registers your IPN URL with Pesapal's API and gives you back the IPN ID to paste into Supabase secrets.

After you get the IPN ID, run:
supabase secrets set PESAPAL_IPN_ID=paste-id-here
supabase secrets set PESAPAL_CONSUMER_KEY=your-key
supabase secrets set PESAPAL_CONSUMER_SECRET=your-secret
